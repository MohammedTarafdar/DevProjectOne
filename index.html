HTML images

HTML images are defined with the img tag. The source file "src", alternative text "alt", "width" and "height" are provided as attributes.

HTML elements

An HTML element is an indivisual component of an HTML document. An element is defined by start tag , some content, and an end tag. Some elements have no content. Those elements are called empty elements. Empty elements do not have an end tag.

Never skip the end tag

Some HTML elements will display correctly, even if you forget the end tag. However, never rely on this! unexpected results and errors may occur if you forget the end tag.

HTML isd not case sensitive

HTML tags are not case sensitive. The HTML standard does not redquire lowercase, but W3C recommends lowercase in HTML and demands lowercase for stricter document types loke XHTML
